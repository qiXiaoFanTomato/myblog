Java2106第三阶段第四周考试内容

数据字典：库名（houses）

  1. 房屋表(house)

    字段名 描述 数据类型 长度 是否主外键 备注
    house_id 编号 int PK 自增
    house_name 名称 varchar 200
    house_pic 图片 varchar 50 图片路径
    house_size 大小 int 例如：15平米
    house_layout 户型 varchar 50 4室1厅
    house_floor 楼层 varchar 50 20/28层
    house_master 户主 varchar 50 石俊
    house_address 位置 varchar 200 洪山-关山 纺织路
    house_estate 小区 varchar 200 金地格林东郡
    house_price 租金 double 1330
    house_type 类型 int FK 引用house_type
    create_date 创建时间 date

  2. 类型表(house_type)

字段名 描述 数据类型 长度 是否主外键 备注
type_id 编号 int PK 自增
type_name 名称 varchar 200 例如：合租
