JQuery

一、jQuery概念

jQuery是一个JS的框架。简化的js的操作。

特点：

1、采用链式结构。

2、支持css操作。

3、拥有丰富的插件。

4、兼容各种主流的浏览器。

5、有丰富的选择器，能够快速的得到要操作的对象。

两种版本：

min版本：项目上线时使用的版本，变量命名极简，没有注释，代码不规范， 能省则省，体积小。

开发调试版本：直接以版本号命名的就是开发调试版本，代码比较规范，注释清晰，变量命名合理，体积较大。作用是在开发调试过程中使用，可以查看源码，或者进行调试。

二、jQuery的安装

两种方式：

1、将该js下载到本地，并在项目中引用。

每次有一个客户端访问服务器，都需要在服务器上下载该js。

2、直接引用网络上的js。(CDN方式)

常用的知名的CDN:

https://apps.bdimg.com/libs/jquery/2.1.4/jquery.min.js

https://lib.sinaapp.com/js/jquery/2.0.2/jquery-2.0.2.min.js

https://ajax.aspnetcdn.com/ajax/jquery/jquery-1.9.0.min.js

https://upcdn.b0.upaiyun.com/libs/jquery/jquery-2.0.2.min.js

https://cdn.staticfile.org/jquery/1.10.2/jquery.min.js

客户端访问服务器，去别的服务器下载js。

三、使用

语法：

$(selector); 得到一个或者一组对象。

$(selector).action()；对象的操作。

四、选择器

在jQuery中使用选择器来获取一个或者一组对象。

4.1 基本选择器

id选择器，相当于document.getElementById(“id”);

语法：$(“#id”);

元素选择器，相当于document.getElementsByTagName(“div”);

语法：$(“div”)

class选择器，相当于document.getElementsByClassName(“c1”);

语法：$(“.c1”);

*：表示得到所有的元素

语法：$(“*”)

,：得到部分元素的组合。

$(“#div1, #div2, .c1, p”); 得到id为div1和div2的元素，以及所有的class为c1的元素以及所有的p标签元素。

4.2 层次选择器

空格 ：得到某个元素的所有后代元素中的某个元素。

例如：$(“#div1 img”) 得到id为div1中的所有的img元素。

>：得到某个元素的所有子代元素中的某个元素。

+ ：得到某个元素后面相邻的兄弟元素

~：得到某个元素的所有兄弟元素

4.3 基础选择器

语法：

$(“:selector”)

:first：表示一组元素中的第一个元素，例如：$(“li:first”) 得到所有li中的第一个li。

:last：表示一组元素中的最后一个元素，例如：$(“li:last”) 得到所有li中的最后一个li。

:even：得到一组元素中的偶数元素，0、2、4等，例如：$(“li:even”) 得到所有li中的偶数下标的li。

:odd：得到一组元素中的奇数元素，1、3、5等，例如：$(“li:odd”) 得到所有li中的奇数下标的li。

:not()：取反，例如：$(“li:not(:first)”) 得到所有li中的除了第一个以外其他的所有li。

:eq()：表示一组元素中的指定个元素，例如：$(“li:eq(0)”) 得到所有li中的第一个（下标为0）li。

:gt()：表示一组元素中下标大于几的所有元素，例如：$(“li:gt(3)”) 得到所有li中的下标大于3的所有li。

:lt()：表示一组元素中下标小于几的所有元素，例如：$(“li:lt(3)”) 得到所有li中的下标小于3的所有li。

4.4 内容选择器

根据内容选择指定的元素。

:contains(text)：得到元素中包含指定文本的内容的所有元素。例如：$(“div:contains(‘武汉’)”); 得到所有的内容包含武汉的div。

:empty：得到元素中内容为空的元素。例如：$(“div:empty”)。得到页面所有空的div。

:parent：得到元素中内容不为空的元素。例如：$(“div:parent”)。得到页面所有不为空的div。

:has(selector)：得到包含有另一个选择器选中内容的所有元素。例如：$(“div:has(:checkbox)”)：得到所有其中包含有checkbox的div。

4.5 可见性选择器

:hidden：得到所有的隐藏元素。例如：$(“div:hidden”); 得到所有的隐藏的div。

:visible：得到所有的可见元素。例如：$(“div:visible”); 得到所有的可见的div。

4.6 属性选择器

通过属性判断。

[attr]：得到包含有某个属性的标签。例如：$(“div[id]”)；得到所有的包含有id的div。

[attr=v]：得到某个属性的值为v的标签。例如：$(“a[title=’a1’]”)；得到所有的所有title值为a1的a标签。

[attr!=v]：得到某个属性的值不为v的标签。例如：$(“a[title!=’a1’]”)；得到所有的所有title值不为a1的a标签。

[attr^=v]：得到某个属性的值以v开头的标签。例如：$(“a[title^=’a1’]”)；得到所有的所有title值以a1开头的a标签。

[attr$=v]：得到某个属性的值以v结尾的标签。例如：$("a[title$='a1']")；得到所有的所有title值以a1结尾的a标签。

[attr*=v]：得到某个属性的值包含v的标签。例如：$(“a[title*=’a1’]”)；得到所有的所有title值包含a1的a标签。

[][][]：对属性做多次限定。例如：$("a[href^='http'][title=a1][class!=c1]")得到所有的href以http开头，title为a1，class不为c1的a标签。

4.7 子元素选择器

:first-child：将得到元素根据父元素不同分为多组，每一组的第一个元素，结果可能是多个元素。与:first不同，:first是指将所有得到的元素归为一组，得到第一个元素，结果是一个元素。

:last-child：将得到元素根据父元素不同分为多组，每一组的最后一个元素，结果可能是多个元素。与:last不同，:last是指将所有得到的元素归为一组，得到最后一个元素，结果是一个元素。

:nth-child：将得到元素根据父元素不同分为多组，每一组指定一个元素，结果可能是多个元素。与:eq不同，:eq是指将所有得到的元素归为一组，得到指定一个元素，结果是一个元素。

注意：nth-child从1开始计算，而eq从0开始计算。

十二、AJAX

语法：

$.ajax({

​ url:””,

​ method:””.

​ dataType:””,

​ contentType:””,

​ data:””,

​ success: function(){},

​ fail:function(){}

});

提供两个简化的版本：

$.get(“url”, {“name”:”mary”, “age”:20}, function(data){}); // 发送get请求

$.post(“url”, {“name”:”mary”, “age”:20}, function(data){}); // 发送post请求

如果没有参数需要传递，也可以不写参数，get方法可以把参数写在url中。

注意：上面的get和post简化的方式，只能发送普通的表单请求，如果要发送json数据，需要使用ajax完全版。

十三、多库共存

当页面上使用多个js的库，并且都使用$()的方式来使用。此时会出现冲突。解决方案如下：

1、直接使用jQuery作为前缀。

2、创建一个新的别名用以在接下来的库中使用jQuery对象。
